    Posting

    Tripcodes

    To use a tripcode, add a # in the name field followed by a password for the tripcode.
    A hash will be applied to the password and the result will be displayed instead.

    To use a secure tripcode, prepend two # instead of one. A secure tripcode will generate a random text that will only be displayed if your password matches.

    Role signatures

    If you wish to display your global or board role when posting, put #rs in the name field. You can't use your role signature and a tripcode at the same time.
    There is a setting in account settings to always display your role signature by default.

    Formatting

    Post quote

    If you write >> followed by the ID number of a post made in the same board, a link to this post will be created.

    >>713705

    Board link

    If you write >>> followed by a board's URI between slashes, a link to that board will be created.

    >>>/furry/

    Cross-board quote

    It's also possible to quote posts from other boards. Just append a post ID number to a board link.

    >>>/cuteboys/15

    Tags

    Greentext

    If you add a > at the start of a line, for some reason the text following it will be green.

    >mfw
    >kek
    >how does this work

    Redtext

    Surround your text with == to make it important. 

    ==I'm so small and insignificant why does no one notice me==

    Hi everyone how's it going? It's beautiful outside!

    Italic

    This place is giving me a feeling of ''deja vu''...

    This place is giving me a feeling of deja vu...

    Underline

    Do __not__ abuse these features or bad things __will__ happen.

    Do not abuse these features or bad things will happen.

    Bold

    '''such wow much text formatting'''

    such wow much text formatting
    (USER WAS BANNED FOR THIS POST)

    Strikethrough

    ~~linux~~ is the best operating system in the world

    linux GNU+Linux is the best operating system in the world

    Spoiler

    Text surrounded by [spoiler] and [/spoiler] tags will be masked until it is hovered over.
    You can't break lines between the two tags. 

    My favorite pokemon is [spoiler]pikachu[/spoiler]

    My favorite pokemon is you____

    Code

    If enabled by the board owner, text surrounded with [code] and [/code] tags will use a monospace font and preserve white-space. 

    if (Tweeter= LynxHub)
  cout << "Block." << endl;
else
  if ( Tweet=Negative)
    cout << "Block" << endl;

    Urls

    Urls beginning with http or https will automatically be converted to hyperlinks.

    https://instagram.com/taylorswift/
